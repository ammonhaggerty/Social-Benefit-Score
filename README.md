# Tech Benefit Score
A simple scoring mechanism for grading the benefit impact of social and media platform.

*Note:* Dualingo is added for white-hat game tactic reference and not meant to be rated as a social or media platform.

## Frature scoring

Each feature is assigned a level of feature intention and recieves points for each.

1. **Strong (✓) = 2 points**
2. **Moderate (•) = 1 point**
3. **Weak (–) = 0 points**

*White Hat vs. Black Hat*
Features are split between **white hat** and **black hat** value, and either receive positive or negative multipliers. The final “Benefit Score” is the sum of PresenceStrength×MultiplierPresence Strength × MultiplierPresenceStrength×Multiplier for each feature, where a **higher (less negative or positive)** total suggests more user-friendly design, and a **lower (more negative)** total suggests heavier reliance on darker engagement tactics.

---

## Feature List & Multipliers

### White Hat (Above “Streaks”)

1. **Focus on societal benefit** → **+3**
2. **Data ownership** → **+2**
3. **Opt-out controls** → **+2**
4. **Badges** → **+1**
5. **Levels** → **+1**

### Black Hat (From “Streaks” Down)

6. **Streaks** → **–1**
7. **FOMO** → **–2**
8. **Infinite scroll** → **–1**
9. **Autoplay** → **–1**
10. **Social pressure** → **–2**
11. **Location sharing** → **–2**
12. **Celebrity/Influencer culture** → **–2**
13. **AI training use (monetizing user content)** → **–3**
14. **Selling your behavior to 3rd party** → **–3**
15. **Misinformation** → **–4**
16. **Mob mentality** → **–5**

**Presence Legend**

- Strong (✓) = 2
- Moderate (•) = 1
- Weak (–) = 0
